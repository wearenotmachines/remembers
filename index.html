Remember a list of items

I'll remember elements with the same label, you can tell me a specific value or I can just remember the content of the element:

<a href="#" data-role="remember" data-remember="shopping">Don't forget to buy these:</a>
<ul>
  <li data-remember-label="shopping" data-remember-value="sweets">Sweets</li>
  <li data-remember-label="shopping" data-remember-value="bread">Bread</li>
  <li data-remember-label="shopping" data-remember-value="cakes">Cakes</li>
  <li data-remember-label="shopping" data-remember-value="apples">Apples</li>
  <li data-remember-label="shopping">Magic wand</li>
</ul>
Don't forget to buy these:
  • Sweets
  • Bread
  • Cakes
  • Apples
  • Magic wand

Remember a bunch of data

I'll remember a data structure made up from different elements - just give me the label and a set of keys and values - or let me remember the element content:

<a data-role="remember" data-remember="unicorn" href="#">Remember this unicorn</a>
<div class="item" data-remember-label="unicorn" data-remember-key="id" data-remember-value="1234">
  <img />
  <ul>
    <li data-remember-label="unicorn" data-remember-key="title">Magical Unicorn</li>
	<li data-remember-label="unicorn" data-remember-key="description">It's basically a horse with a plastic horn glued to it.</li>
	<li>
	  <label>What colour would you like?</label>
	  <select data-remember-label="unicorn" data-remember-key="colour">
	    <option value="white" selected>White</option>
	    <option value="green">Green</option>
	    <option value="blue">Blue</option>
	  </select>
	</li>
	<li>
	  <label>Name your unicorn:</label>
	  <input type="text" data-remember-label="unicorn" data-remember-key="name" placeholder="choose a name for your magical beast" />
	</li>
  </ul>
</div>
Remember this unicorn
  • Magical Unicorn
  • It's basically a horse with a plastic horn glued to it.
